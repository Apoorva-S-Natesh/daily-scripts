    Arrow Functions:

    • If exactly one parameter, then no need of wrapping paranthesis: username => { ... }
    • If no pararmeters or more than one, then you cannot omit the paranthesis: () => { ... } or (userName, message) => { ... }
    • Omitting function body curly braces if there is no other logic other than a return statement inside the function: number => number * 3;
    • If using the above shorter alternative and want to return a JavaScript object then need to wrap the object in an extra pair of parentheses: number => ({ age: number });
